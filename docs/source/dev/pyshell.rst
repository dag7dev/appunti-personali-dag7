Pyenv consente di avere ambienti virtuali sulla propria macchina.

Installazione
-------------
* installare ``pip``
* lanciare il comando ``pip install pipenv``
	* in caso di problemi ``sudo -H pip install -U pipenv``

Utilizzo
--------
* in una cartella lanciare ``pipenv shell``
